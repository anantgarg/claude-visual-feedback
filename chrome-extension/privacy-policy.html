Privacy Policy

Last updated: March 20, 2026

Overview

Visual Feedback for Claude Code is a Chrome extension that enables developers to leave visual feedback on websites. All data stays on your local machine.

Data Collection

This extension does not collect, transmit, or store any personal data on external servers. Specifically:

  • No data is sent to any third-party servers
  • No analytics or telemetry is collected
  • No cookies are set by the extension
  • No user accounts or authentication are required

Local Communication

The extension communicates exclusively with a local server running on your machine (localhost/127.0.0.1). This server is started by you within your development environment. The communication includes:

  • Feedback comments you write
  • Screenshots of selected page areas
  • CSS selectors and coordinates of selected elements
  • Page URLs you provide feedback on

All of this data is stored as files on your local filesystem, in your project directory.

Permissions

  • activeTab: Used to interact with the current page for element selection and screenshot capture
  • storage: Used to remember your preferences (server connection, author name) locally in Chrome
  • scripting: Used to inject the feedback UI into web pages
  • host_permissions (localhost): Used to communicate with the local feedback server only

Screenshots

Screenshots are captured only when you explicitly select an element and submit feedback. They are stored locally on your machine and are never transmitted externally.

Contact

For questions about this privacy policy, open an issue on the project's GitHub repository.
